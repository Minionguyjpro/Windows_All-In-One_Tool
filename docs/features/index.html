Features of Windows All-In-One Tool

Welcome to GitHub Pages.

Function Explanation
DISM and SFC Repair Tries to repair corrupted files on the computer using DISM and SFC.
Create Restore Poin Creates a Windows restore point that can be restored later on if it is needed.
Windows IP Configuration Shows Windows current IP configuration, including network adapters and IP address.
Check Windows Version Shows the current Windows version (including build and Windows version and edition).
Ping Tool Allows you to ping a website to see whether it is up/responding or not.
Delete DNS cache Delete (flush) DNS cache of the computer.
Open Terminal Open Windows Command Prompt or PowerShell easily to run certain commands.
Credits Shows the credits and contributors for making the tool possible.
Tool Information Shows some useful information about the tool, such as the author and version.
Exit Closes the tool.
Boot into BIOS/UEFI (WinUEFI) Boots the computer into the BIOS/UEFI settings easily. This is based on <a href="https://github.com/FreakinSoftMania/WinUEFI">WinUEFI</a>.
Show System Information Shows some very useful information about the computer, such as the host name and OS version.
Show Windows Directory Tells in which directory/folder Windows is currently installed.
Empty temporary folder Clears the temporary folder with %TEMP% and %TMP% variable, so that you can free space and programs can build new cache.
Check Windows last boot time Shows the date and time that the system was booted up for the last time.
Show Wi-Fi Password Shows your Wi-Fi Password. NOTE: please make sure no other person can access your computer, or only trusted people.
Install/Update Programs Allows you to easily install or update programs using Windows built-in <code>winget</code> command.

Rather Drive Stick?
