Features of Windows All-In-One Tool

Features

Function Explanation
1. DISM and SFC Repair Tries to repair corrupted files on the computer using DISM and SFC.
2. Create Restore Point Creates a Windows restore point that can be restored later on if it is needed.
3. Windows IP Configuration Shows Windows current IP configuration, including network adapters and IP address.
4. User List Shows all the users that exist on the computer.
5. Check Windows Version Shows the current Windows version (including build and Windows version and edition).
6. Ping Tool Allows you to ping a website to see whether it is up/responding or not.
7. Delete DNS cache Delete (flush) DNS cache of the computer.
8. Open Terminal Open Windows Command Prompt or PowerShell easily to run certain commands.
9. Credits Shows the credits and contributors for making the tool possible.
10. Tool Information Shows some useful information about the tool, such as the author and version.
11. Exit Closes the tool.
12. Boot into BIOS/UEFI (WinUEFI) Boots the computer into the BIOS/UEFI settings easily. This is based on WinUEFI
13. Show System Information Shows some very useful information about the computer, such as the host name and OS version.
14. Show Windows Directory Tells in which directory/folder Windows is currently installed.
15. Empty temporary folder Clears the temporary folder with %TEMP% and %TMP% variable, so that you can free space and programs can build new cache.
16. Check Windows last boot time Shows the date and time that the system was booted up for the last time.
17. Show Wi-Fi Password Shows your Wi-Fi Password. NOTE: please make sure no other person can access your computer, or only trusted people.
18. Install/Update Programs Allows you to easily install or update programs using Windows built-in winget command.
